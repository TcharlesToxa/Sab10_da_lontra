Adobe DreamWeaver

O que é e para que serve o DreamWeaver

É um programa de webdesign que combina uma interface visual com um editor de código, permitindo a criação dinâmica de sites, oferecendo suporte a HTML, CSS, JavaScript e importação de atividades de biblioteca.

Podemos desenvolver: Páginas de comércio eletrônico, Blogs, Boletins informativos, Portfólios, Aplicativos web para dispositivos móveis, Formulários web, Wireframes, Protótipos, Elementos interativos
